# Castaway

Castaway problem.

## Requirements

- Gym
- Baidu PARL

## Usage

### Train the model

```python
python3 main.py --task=train
```

A `dqn_model.ckpt` file will be outputed in the `./results` directory.

### Plot the results

```python
python3 main.py --task=plot
```
